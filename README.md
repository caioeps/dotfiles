## Vim config file

This .vimrc file uses Vundle to control its packages.

### 1) Installing Vundle
```
$ git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

### 2) Get Started
1. Open Vim
```
$ vim
```
2. Inside Vim run the following command
```
:PluginInstall
```
